== README

This is a simple Blogging application developed with Ruby on Rails. It displays basic CRUD functionality.

This application requires a local Rails framework on any machine you wish to run it on. 

To Run:

1. Download the .zip folder from Github
2. Copy the contents into a new folder 
3. Move the new folder into your local Rails project directory
4. Run your Rails server from the Rails command prompt
5. In your web browser type in the URL: http://localhost:3000/articles

Have Fun!
